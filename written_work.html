I used CSS to fix my image to the background and used attributes such as background: center, attachment, repeat and size to centre the image, fix it in place, prevent it from repeating and thus creating a tiled effect and size to make the image responsize to changes in browser windows or mobile browsers.

I made the links appear horizontally along the page, removed the underline using the text-decoration: none; attribute and used the hover: attribute to increase the size of the font when the user hovers their mouse over the link.
